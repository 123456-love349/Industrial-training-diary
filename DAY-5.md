Day 5 – Large Language Models (LLMs) and Ollama
Objective

Today's session focused on understanding Large Language Models (LLMs), how they work, and how to run open-source AI models locally using Ollama.

1. Large Language Models (LLMs)

Large Language Models are advanced deep learning models trained on massive datasets of books, articles, websites, code repositories and conversations.

They can understand, generate and analyze natural language.

Popular LLMs include:

ChatGPT (OpenAI)
Qwen
Llama
Mistral
Gemma
DeepSeek

4. Ollama

Ollama is a tool that allows developers to run open-source LLMs locally on their own computer without relying on cloud APIs.

Benefits:

Offline usage
Better privacy
No API cost
Faster local experimentation
Easy model management

Supported models include:

Llama
Mistral
Gemma
Qwen
DeepSeek
5. Ollama Installation

We learned how to install Ollama and verify the installation.

General workflow:

Install Ollama.
Download a model (e.g., Llama or Qwen).
Run the model locally.
Interact with the model using the terminal or integrate it into applications through its local API.
