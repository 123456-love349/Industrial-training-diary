Day 6 – Public MCP Servers, GLM, NVIDIA Omniverse, Docker and AWS Project
Objective

The objective of today's session was to understand how AI applications can use public MCP servers, explore GLM models, learn about NVIDIA Omniverse for simulation, understand Docker containers, and build a basic cloud project using Amazon Web Services (AWS).

1. Public MCP Server

A Public MCP Server is an MCP server that exposes AI tools and services over a standardized protocol. Instead of building every tool locally, developers can connect AI models to publicly available MCP servers to access external functionalities.

Advantages
Reusable AI tools
Standardized communication
Easy integration
Faster development
Scalable architecture
Applications
File Management
Database Access
Browser Automation
Web Search
External APIs
2. GLM (General Language Model)

GLM (General Language Model) is a family of Large Language Models designed for understanding and generating human language. Like other LLMs, GLM can perform tasks such as:

Question Answering
Text Generation
Translation
Summarization
Code Generation

The session focused on understanding how GLM models compare with other open-source LLMs and how they can be integrated into AI applications.

NVIDIA Omniverse is a platform for building and simulating 3D environments using AI and digital twins.

Applications

Digital Twins
Autonomous Vehicle Testing
Features
Real-time collaboration
AI-powered simulations
Physics-based rendering
Multi-user environment
Integration with AI models
4. Docker

Docker is a containerization platform that packages an application along with all its dependencies into a single container, ensuring consistent execution across different systems.

Why Docker?

Without Docker, applications may fail due to missing libraries or different operating system configurations.

Docker solves this problem by creating isolated containers.

Benefits
Easy deployment
Consistent environment
Faster application setup
Better scalability
Simplified dependency management
5. AWS (Amazon Web Services)

Amazon Web Services (AWS) is one of the world's leading cloud computing platforms. It provides services for hosting applications, storing data, deploying AI models, and managing cloud infrastructure.

Some commonly used AWS services include:

Amazon EC2
Amazon S3
AWS Lambda
Amazon RDS
Amazon SageMaker
